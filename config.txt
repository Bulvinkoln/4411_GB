Основыне команды гит 
git init- запуск гита 
git add  название файла  - добавить файл в гит.
git add .  добавить все вайлы в паке в гит. 
git commit -am добавить файл и к нему комментарии. 
clear - очистить всю командную строку 
git log  - отображение всех изминении 
git log --online  отображение всего журнала изминении минималистично с отоброжениями ветвями, если таковы присутсвуют. 
git checkou  перемещение между версиями или ветвями 
git diff  -показ разницы 
git branch  создание ветки 
git checkout -b название ветки  - создание ветки и переход в нее
git reset --hard  - полностью отменякт коммит 
git merge --abort  - отменяет конфликты 
git clone  ссылка - клонирование репозитория
git pull   обновить свой репозитории до последней версии, если он обновился на сайте  облака
git push отправить репозитории
